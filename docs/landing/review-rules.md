# Правило ревью текстов перед публикацией

Перед публикацией любых изменений в текстах лендинга обязательно пройти этот чек-лист:

1. Обновить тексты только через `web/content/landing-content.json`.
2. Проверить соответствие формулировок ограничениям из `TZ.md` (разделы 2.1–2.3).
3. Запустить статическую проверку:
   ```bash
   python scripts/check_landing_content.py
   ```
4. Проверить, что до перехода в оплату есть два явных блока:
   - «Не является консультацией/прогнозом»;
   - «Возвратов нет».
5. На ревью приложить результат команды проверки и кратко описать изменения текста.

Публикация без прохождения этого чек-листа запрещена.
